# 🌹 Hammer the Rose

A fun, addictive arcade game built as my **first GitHub project**.

![Game Screenshot](https://rleo0069-del.github.io/Hammer-the-rose/preview.png)

## 🎮 How to Play
- **Move**: Left / Right arrows or A / D
- **Swing Hammer**: SPACE
- **Pause**: P
- **Music**: M
- Collect 🌹 **Roses** for points
- Smash 🌵 **Thorns** for big bonus + combo
- Don't get hit!

## ✨ Features
- Combo multiplier system
- Progressive difficulty
- Sound effects + background music
- Mobile touch support
- High score saving
- Share your score!
